# 🛣️ RoadSense India
### Real-Time Road Safety & Accident Hotspot Intelligence System

A complete Data Science internship project featuring EDA, ML modelling,
IoT simulation, and an interactive Streamlit dashboard.

---

## 🗂️ Project Structure
```
RoadSense-India/
├── data/
│   ├── accidents_raw.csv      ← download from Kaggle (see below)
│   ├── cleaned.csv            ← generated by EDA notebook
│   └── summary_stats.csv      ← generated by EDA notebook
├── notebooks/
│   └── eda.py                 ← Full EDA (paste into Jupyter cells)
├── app/
│   ├── app.py                 ← Streamlit dashboard (main file)
│   ├── ml_model.py            ← Linear Regression model
│   └── iot_simulator.py       ← IoT highway sensor simulator
├── requirements.txt
└── README.md
```

## ⚙️ Setup

### 1. Clone or create the folder
```bash
mkdir RoadSense-India && cd RoadSense-India
```

### 2. Create virtual environment
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Get the dataset
Download from Kaggle:
https://www.kaggle.com/datasets/saurabhshahane/road-accident-india

Rename the file to `accidents_raw.csv` and place it in the `data/` folder.

---

## 🚀 Running the Project

### Step 1 — Run EDA notebook first (generates cleaned.csv)
Open `notebooks/eda.py` in VS Code as a Jupyter notebook and run all cells.

### Step 2 — Launch the dashboard
```bash
streamlit run app/app.py
```

Opens at: http://localhost:8501

## 📊 Features
- **5-tab interactive dashboard** (Overview, State Analysis, ML, IoT, Insights)
- **State & year filters** applied across all charts
- **6 EDA charts** with written insights
- **Linear Regression** model predicting accident counts (R² shown live)
- **Real-time IoT simulator** with risk scoring and alert cards
- **Key findings** panel summarising actionable insights

---

## ☁️ Deployment (Streamlit Cloud)
1. Push this repo to GitHub (public)
2. Go to https://share.streamlit.io
3. Connect your GitHub → select `app/app.py` → Deploy
4. You get a URL like: `yourname-roadsense.streamlit.app`
